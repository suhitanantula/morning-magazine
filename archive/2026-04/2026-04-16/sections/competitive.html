MetaGPT has emerged as the highest-rated multi-agent framework on GitHub with 67,091 stars, simulating entire software companies through role-playing agent teams.

The v0.8.2 release demonstrates growing enterprise interest in AI agents that can collaborate on complex workflows.

This signals intense competition for The Helix Lab's workflow automation ambitions as developers increasingly adopt multi-agent architectures.

CrewAI has exploded to 48,935 stars, becoming the fastest-growing multi-agent framework in the ecosystem with release 1.14.2a4.
